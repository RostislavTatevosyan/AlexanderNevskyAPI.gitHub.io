Alexander Nevsky API

Alexander Nevsky API это бесплатный JSON API для получения коровых высказываний великого актера Александра Невского (ie. Мистер Пельменная)

Использование

Получите случайную шутку в формате JSON.

http://nevskyapi.online/statements/random

Пример ответа:

        
            {

                
                "valueRu": "Абсолютли!", "valueEn": "Absolutely!""
            

            }
